About
=====

A low-level network/socket library.


Installation
============

Lastest Git Revision
--------------------

With CMake:

	$ git clone git://github.com/Neopallium/lua-llnet.git
	$ cd lua-llnet ; mkdir build ; cd build
	$ cmake ..
	$ make
	$ sudo make install


Example usage
=============

	local llnet = require"llnet"
